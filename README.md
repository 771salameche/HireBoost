# HireBoost 🚀

**HireBoost** is a premium, AI-powered career platform designed to empower job seekers by bridging the gap between their current skills and their dream roles. Built with a focus on speed, precision, and high-end aesthetics, HireBoost transforms the job search from a daunting task into a strategic journey.

## ✨ Features

### 🔍 CV Analyzer
Upload your resume and receive instant, AI-driven feedback. Identify key strengths, surface critical weaknesses, and get actionable suggestions to make your profile stand out to recruiters.

### 🎯 Job Matcher
Paste a job description and let HireBoost calculate your compatibility score. Understand exactly how well your experience aligns with specific roles before you apply.

### 🗺️ Skill Gap Roadmap (New!)
The core of your professional growth. HireBoost analyzes job requirements against your CV to generate a **prioritized 3-week learning plan**.
- **Weekly Focus:** Structured milestones (e.g., Next.js, System Design).
- **Curated Resources:** Direct links to high-quality documentation and courses.
- **Priority Indicators:** Focus on what matters most for the hire.

### ✍️ AI Generator
Effortlessly generate tailored cover letters and optimized CV sections tailored to specific job descriptions using advanced AI prompts.

### 🎙️ Interview Simulator
Practice your pitch with our interactive simulator. Get ready for tough questions with AI-generated scenarios based on your target industry.

## 🛠️ Tech Stack

- **Frontend:** React 18 (Vite) + TypeScript
- **Styling:** Tailwind CSS + Shadcn UI (Radix UI Primitives)
- **Animations:** Framer Motion (Micro-interactions & Smooth Transitions)
- **Icons:** Lucide React
- **Routing:** React Router DOM v6
- **State Management:** TanStack Query (React Query)
- **Validation:** Zod + React Hook Form
- **Testing:** Vitest + React Testing Library

## 🎨 Design System: "Obsidian Gold"

HireBoost uses a bespoke "Obsidian Gold" design system to provide a premium, executive feel:
- **Primary Palette:** Obsidian Dark (`#0a0a0c`) with Champagne Gold accents.
- **Glassmorphism:** Heavy use of `premium-glass` utilities and sub-pixel borders.
- **Typography:** `Geist` for sharp, sophisticated headings and `Inter` for maximum readability.

## 🚀 Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Installation
1. **Clone the repository:**
   ```bash
   git clone https://github.com/771salameche/HireBoost.git
   cd HireBoost
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```

## 🧪 Testing

Run the test suite using Vitest:
```bash
npm run test
```
